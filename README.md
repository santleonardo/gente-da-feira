# 🎡 Gente da Feira 

O **Gente da Feira** é um PWA (Progressive Web App) de utilidade pública e rede social regional, desenvolvido exclusivamente para os moradores e empresas de **Feira de Santana - BA**.

## 🚀 O Projeto
Diferente das redes sociais globais, o Gente da Feira foca no que acontece "da porta para fora". É uma ferramenta para conectar cidadãos, microempreendedores (MEIs), profissionais liberais (médicos, professores, autônomos) e grandes atacadistas da região.

### 🌟 Diferenciais
- **Hiper-Localidade:** Feeds divididos por eixos geográficos (Centro, Tomba, SIM, Cidade Nova, etc.).
- **Confiança:** Sistema de verificação manual de perfis profissionais e comerciais para evitar golpes.
- **Utilidade:** Foco em avisos imediatos (vagas de emprego, ofertas relâmpago, trânsito e utilidade pública).
- **Comunidade:** Salas de bate-papo moderadas pelos próprios donos (empresas e profissionais).

## 🛠️ Tecnologia
Este projeto foi construído para ser leve, rápido e acessível, utilizando:
- **Frontend:** HTML5, CSS3 (Tailwind CSS) e JavaScript Puro.
- **Backend:** Supabase (Banco de dados e Realtime).
- **Hospedagem:** GitHub Pages.
- **Conceito:** PWA (não precisa baixar na loja, instala direto pelo navegador).

## 📈 Status do Projeto
Atualmente em fase de **MVP (Mínimo Viável)**, focado em validar a interação entre os primeiros 1.000 usuários em bairros específicos.
